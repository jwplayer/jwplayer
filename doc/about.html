About JW Player

The JW Player is the world's most popular HTML5/Flash embeddable media player for websites. It supports a wide array of (mobile) platforms and media formats. It is open-source and easy to configure, customize and extend.

Browser & Media Support

Since the JW Player leverages both HTML5 <video> and Adobe Flash, it supports a wider range of browsers/devices and audio/video formats than is possible with HTML5 or Flash alone. Here is an overview:

Features & Editions

JW Player is a mature product with a complete and robust set of features:

  • A slick and intuitive interface that works on all popular browsers and devices.
  • An elegant and rich playback engine for rendering your media content in HTML5 and/or Flash.
  • An extensive list of options for easily configuring layout and behavior of the player.

JW Player can be customized and extended beyond its existing feature set. A PNG Skinning Model is available to completely match the player's look and feel to a website and a JavaScript API is available to add new functionalities or build rich player-page interactions.

Advanced Features

JW Player 6 includes support for many advanced use cases not commonly found in other players:

  • Comprehensive playlist support for loading multiple media items, file formats and quality levels.
  • Display of captions or subtitles using external WebVTT, SRT, DFXP files or embedded MP4 tracks.
  • Live and adaptive streaming using the Adobe RTMP protocol, plus pseudo-streaming in Flash.

Extensive integrations with the popular Drupal CMS and WordPress CMS are available. They enable easy setup of JW Player and embedding of audio/video content into your site.

Player Editions

JW Player 6 is available in four distinct editions:

Free
This edition is free for non-commercial use under a Creative Commons license. It includes all features listed above. During playback, a small JW Player logo is displayed in the top right corner of the display.
Pro
This edition contains the same features as the Free edition, but has no restriction around commercial use. It also does not display a JW Player logo. Instead, you can set your own.
Premium
This edition contains all features of the Pro edition, plus adaptive streaming using Apple HLS in Flash, overlays for Related Videos and Social Sharing and an integration with Google Analytics. It also includes 8 premium skins.
Ads
This edition contains all features of the Premium edition, plus video advertising support using Direct campaigns, Google's IMA or IAB-standardized VAST/VPAID.
