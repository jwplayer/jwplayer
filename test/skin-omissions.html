Skin Omissions

List of test devices

Player skins can leave out certain elements. This page tests skins that omit various elements.

No components

ActionExpected result(s)Works?

Start playback The Five controlbar and Six dock should display
Pause playback, then hover the controls The Six time/button/hd/volume tooltips should display.
