Class: InstreamAdapter

InstreamAdapter(_controller, _model, _view, _mediaPool)

new InstreamAdapter(_controller, _model, _view, _mediaPool)

InstreamAdapter JW Player instream API. Instantiated via jwplayer().createInstream(). Only one instance can be created per player. It is destroyed via jwplayer().instreamDestroy().

Parameters:
Name Type Description
_controller Controller

The player controller instance
_model Model

The player model instance
_view View

The player view instance
_mediaPool MediaPool

The player media pool
